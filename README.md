# thearft

Personal site of Md. Arafat Hossain — Head of Imports & Technology, Al-Amin
Trading Organization. Narrative site, not a job-seeker portfolio: suppliers
first, automation clients second.

Read `CLAUDE.md` first — it is the project constitution (brand, rules, copy,
design, stack). Copy sources live in `docs/copy/`; the visual direction is the
Claude Design handoff in `docs/design/`.

## Stack

Next.js 15 (App Router) · TypeScript (strict) · Tailwind CSS v4 · static
export for GitHub Pages (`thearft.github.io`).

## Commands

```sh
npm run dev     # local dev server
npm run build   # static export → out/
npm run lint    # eslint
```
